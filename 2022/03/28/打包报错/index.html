vue run build时的错误解决

报错信息

Module not found: Error: Can’t resolve ‘@babel/runtime/helpers/esm/extends in ‘D:\work\海鑫\代码\hisignDDCT\src\components\hisign\utils’

将此信息与网上检索工具搜索到的结果皆与本项目无关，大多是指history依赖版本错误的问题，无法解决此问题，后经排查、尝试等操作

  1. 删除node_modeuls/目录
  2. 删除package.lock.json 此文件会固化会固化当前安装的每个软件包的版本
  3. 重新安装依赖后打包成功
