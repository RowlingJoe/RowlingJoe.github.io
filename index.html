weex的简单上手

  1. 参考文档 weex开发文档
  2. 安装weex-toolkit
     npm install weex-toolkit -g
 //升级weexpack
 weex update weexpack@1.2.1
  3. 创建项目
    weex create helloWorld
  4. 进入项目并安装依赖运行
    cd helloWorld
npm install
npm start
  5. 添加IOS或Android平台
    weex platform add android
  6. 编译运行

    为了能在本地机器上打开 Android 和 iOS 项目，应该配置好客户端的开发环境。对于 iOS，你应该安装并且配置好 Xcode。对于 Android，你应该安装并且配置好 Android Studio。当开发环境准备就绪后，运行下面的命令，可以在模拟器或真实设备上启动应用：

    weex run android
