获取form表单里的各项数据

使用jquery的serialize获取到url然后使用getUrlVars方法将url可以解析为对象，在通过decodeURI对编码的汉字进行解码
$(“form”).submit(function(event) {
event.preventDefault()
var fields = $(“form”).serialize();
var jsonStr = getUrlVars(fields)
//这个就是你的得到的json对象然后$.ajax传过去
for(var k in jsonStr) {
jsonStr[k] = decodeURI(jsonStr[k])
}
jsonStr = JSON.stringify(jsonStr)
})

按位运算符（&，|， ^， <<，>>，>>>，~）
| 运算符 | 用法 | 描述 |
| ————- |:————-:| —–:|
| 按位与 | a&b | 对于每一个比特位，只有两个操作数相应的比特位都是1是结果才位1，否则为0 |
| 按位或 | a|b | 对于每一个比特位，当两个操作数相应的比特位至少有一个1时。结果为1，否则为0 |
| 按位异或 | a^b | 对于每一个比特位，当两个操作数相应的别特为有且只有一个1是结果为1，否则为0 |
| 按位非 | ~a | 反转操作数的比特位，0变1；1变0 |
| 左移 | a<<b | 将a的二进制形式向左移b（<32）比特位，右边用0填充 |
| 有符号右移 | a>>b | 将a的二进制形式向左移b（<32）比特位，丢弃被移出的位|
| 无符号右移 | a>>>b | 将a的二进制形式向左移b（<32）比特位,并使用0在左侧填充 |

获取数组里指定元素的索引

Array.prototype.indexOf = (val, key) => {
// 判断是不是对象
if (typeof val != ‘object’) {
for (let x = 0; x < this.length; x++) {
if (this[x] === val) return x
}
return -1
} else {
for (let x = 0; x < this.length; x++) {
if (this[x][key] === val) return x
}
return -1
}
}
